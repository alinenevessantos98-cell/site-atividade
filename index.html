Sobre o Site

Este site foi criado para acompanhar a evolução do aluno em um curso online. Ele mostra conteúdos estudados, progresso e informações importantes, funcionando tanto em computador quanto em celular.

Progresso do Aluno

  • HTML e CSS – Em andamento
  • JavaScript – Não iniciado

Conceitos de Internet

O protocolo HTTP é usado para comunicação entre navegador e servidor, utilizando portas 80 (HTTP) e 443 (HTTPS), com segurança através de TLS. Já o SMTP é usado para envio de e-mails, normalmente nas portas 25 ou 587, enquanto POP3 e IMAP são usados para recebimento das mensagens.

Cache e Desempenho

O cache permite que páginas já visitadas carreguem mais rápido, armazenando dados temporários no navegador. Isso pode ser controlado por políticas como cache-control e expires.
